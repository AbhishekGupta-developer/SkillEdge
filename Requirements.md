# Requirements

* Refactor JWT implementation
* Build it on MS architecture

1. Authentication & User Management
   * Student registration & login
   * Admin login
   * Teacher login
   * Forgot password (OTP / Email)
   * Role-based access (Admin / Teacher / Student)
   * JWT-based authentication
   * Profile management (photo, phone, address)

2. Course & Batch Management
   * Create courses (e.g., Java, DSA, Python, Drone Technology)
   * Create/manage batches
   * Assign teachers to batches
   * Batch schedule & timetable
   * Student batch enrollment

3. Attendance System
   * Daily attendance for each batch
   * Teacher can mark attendance
   * Student can view attendance report
   * Export attendance as Excel/CSV

4. Online Tests & Evaluation
   * Add MCQ/Subjective tests
   * Auto-evaluation for MCQ
   * Manual evaluation for subjective
   * Result generation & student performance chart

5. Assignment Submission System
   * Teachers upload assignments 
   * Students upload assignment files 
   * Automatic deadline check 
   * Late submission flag

6. Payment / Fee Management 
   * Fee structure for each course 
   * Student fee status 
   * Online/Offline payment record 
   * Invoice / receipt generation

7. Notices & Announcements
   * Admin can post notices 
   * Show notices on student dashboard 
   * Push/email/WhatsApp notification

8. Study Material & Resources
   * Teachers upload PDFs, videos, links 
   * Students download/stream content 
   * Folder-wise organized content

9. Certifications & Completion Tracking
   * Auto-generate course completion certificates 
   * Generate certificate with QR verification 
   * Student progress tracking

10. Placement Portal
    * Post jobs/internships 
    * Students apply 
    * Resume upload 
    * Shortlisting system
